| Command                 | Meaning                                       |
| ----------------------- | --------------------------------------------- |
| `python script.py`      | Run a Python file                             |
| `python -m pip`         | Run the `pip` module                          |
| `python -m venv .venv`  | Create a virtual environment                  |
| `python -m http.server` | Start a simple web server                     |
| `python -c "..."`       | Execute Python code from the command line     |
| `python -i`             | Start interactive mode after running a script |
| `python --version`      | Show Python version                           |
| `python --help`         | Show all command-line options                 |
